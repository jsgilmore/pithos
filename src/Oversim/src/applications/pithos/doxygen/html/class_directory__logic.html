Detailed Description

The Directory logic class, which is used by the Directory Server Omnet module implements P2P bootstrapping of Pithos. It provides a known IP address, where Super Peers may register their IP addresses and locations. Peers can then query this service and probide their own location whereupon the server will respond with the IP of a Super Peer nearest to that location.
