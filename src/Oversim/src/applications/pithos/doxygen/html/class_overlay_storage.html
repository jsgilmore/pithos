Detailed Description

The OverlayStorage module is responsible for handling all aspects of overlay storage on the sending side. When another module wants to store an object in overlay storage, that module sends the object to this module. The reason for this indirection is that it allows for a modular design of overlay storage and the further implementation of other types of overlay storage.
